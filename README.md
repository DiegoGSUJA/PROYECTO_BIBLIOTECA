# Gestor de Biblioteca Digital

Proyecto educativo en C++ que simula la gestión de una biblioteca digital, poniendo en práctica estructuras de datos (listas, mapas, vectores) y buenas prácticas de control de versiones con Git y CMake.

## Estructura del proyecto

- `/src`: Código fuente del proyecto
- `/test`: Pruebas unitarias
- `/diagrams`: Diagramas UML y documentación técnica
